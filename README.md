# Practice_09_cpp
Определить структурный тип, описывающий расписание маршрутов электропоездов
(название маршрута, пункт отправления, пункт прибытия, время отправления, время
прибытия, количество вагонов, тип электропоезда: товарный, пассажирский,
ремонтный и др.). Заполнить структурный массив 20-ю записями. Переписать из
исходного массива в другой массив, информацию только о тех грузовых маршрутах,
что идут в приморские города. Затем новый массив отсортировать по количеству
вагонов. Вывести все данные по конкретному электропоезду. Вывести 3 самых
длинных пассажирских поезда. Реализовать функцию изменения данных о
маршруте. Реализовать вывод отфильтрованных данных в виде оберточной
функции.
